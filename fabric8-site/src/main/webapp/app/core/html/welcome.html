fabric8 is an open source integration platform

Features

fabric8 makes it really easy provision, configure and manage open source integration technologies on a number of containers (or JVMs).

watch a demo » or check out more demo videos

fabric8 has been designed to be the best platform to run techologies like: Apache ActiveMQ, Apache Camel, Apache CXF, Apache Karaf, Apache Tomcat, Apache ServiceMix

About

fabric8 is the open source community project (sponsored by Red Hat, Inc.) for the Fabric capability which has been integrated into JBoss Fuse and JBoss A-MQ for a number of years (since 6.0 of JBoss Fuse and 7.x of Fuse ESB)

the new io.fabric8 package structure is included in version 6.1 or later of JBoss Fuse and JBoss A-MQ
